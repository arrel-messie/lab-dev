# Architecture Microservices

## Principes appliqués
- Découpage par domaine métier
- Base de données par service
- Communication asynchrone
- API Gateway
- Service Discovery

## Patterns implémentés
- Circuit Breaker
- CQRS
- Event Sourcing
- Saga Pattern
- Backend for Frontend

## Technologies utilisées
- Spring Cloud
- Kafka
- Docker
- Kubernetes
- Service Mesh
